# ARCSIX 2024 Data Map

Interactive HTML reference for the [ARCSIX](https://espo.nasa.gov/arcsix) (Arctic Radiation–Cloud–Aerosol–Surface Interaction eXperiment) 2024 airborne campaign.

## What it is

A single-page instrument and data inventory covering all three platforms (NASA P-3B, NASA G-III, SPEC Learjet), organised by science category:
- Navigation & Meteorology
- In-Situ Aerosol Properties
- In-Situ Cloud & Precipitation Microphysics
- Trace Gases & Water Vapor
- Remote Sensing & Radiation (Active lidar/radar · Passive Remote Sensing · Radiation)
- Surface & Ground-Based
- Satellite & Complementary Datasets

Each instrument row shows platform badges, EarthData / DOI links, video links (data workshop talks), and an expandable key-variables panel with uncertainty estimates cross-referenced to the ARCSIX ESSD paper.

## How to use

Open `index.html` in any browser — no server needed.  
Click section headers to expand, click **▶ Variables** to see variable tables, hover ⚠️ flags for cells where uncertainty info is still needed.
